# ⚙️ Installation Guide

## 📥 Clone Repository

```bash
git clone https://github.com/codrithm-dev/codrithm_zimran.git
```

---

## 📂 Navigate Into Folder

```bash
cd codrithm_zimran
```

---

## 📦 Install Dependencies

```bash
npm install
```

---

## 🚀 Start Development Server

```bash
npm run dev
```
